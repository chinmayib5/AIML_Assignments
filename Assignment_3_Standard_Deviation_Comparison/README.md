# Assignment 3 – Standard Deviation Comparison

This assignment compares three models with the same mean but different standard deviations.

## Given Values
- Mean (μ) = 55
- σₐ = 4
- σᵦ = 10
- σ𝒸 = 15

## Objective
To determine which model is better based on variability and data spread.

## Conclusion
Model A (σ = 4) is the best because it has the lowest standard deviation, indicating higher consistency and stability.
